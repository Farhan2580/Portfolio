## 💎 Key Features

* **V15 Neural Network Menu:** A custom-designed 3D perspective navigation panel that scales seamlessly across both ultra-wide screens and mobile viewports.
* **Smart Page Color Switcher:** An integrated Dark/Light mode toggle positioned neatly beside the menu that modifies the body color dynamically while keeping a consistent dark theme on the navigation bar.
* **Embedded AI Assistant Chatbot:** A functional, smart virtual assistant component to provide instant contextual data about my background, skills, and contacts.
* **Glassmorphism Design:** Subtle blurring effects, micro-interactions, gradient reveals, and high-quality typography tracking for a premium desktop experience.
